# 💡 灵感库 (Inspiration)

这里记录爸爸妈妈的灵光一现。把碎片化的想法记下来，等时机成熟时再转化为完整的需求/PRD。

---

## 📖 怎么使用

### 1. 记录灵感
随时把灵感发给阿布，阿布会帮你整理成 `.md` 文件放在这里。

### 2. 定期回顾
有空时浏览 `inspiration/` 目录，看看哪些想法值得深入。

### 3. 转化为需求
选定一个灵感后，阿布会帮你：
- 完善细节
- 写 PRD
- 走 PR 流程实现

---

## 📝 灵感模板

每个灵感文件遵循以下结构：

```markdown
# 灵感：XXX 简短标题

## 📋 基础信息
- **日期**：YYYY-MM-DD
- **来源**：URL / 产品名 / 文章标题
- **记录人**：爸爸/妈妈
- **标签**：#功能 #技能 #自动化 #AI #金融 ...

## 💡 原始灵感
（当时看到什么，想到了什么，原话记录）

---

## 🔍 深度分析

### 解决什么问题
- 痛点是什么
- 现有方案的不足

### 目标用户/场景
- 谁会用
- 在什么场景下用
- 使用频率（每天/每周/偶尔）

### 商业价值
- 能省时间/省钱/赚钱吗
- 市场规模/竞品情况
- 变现可能性

### 技术可行性
- 需要什么技术栈
- 现有 skills 能否复用
- 预计开发工作量（小/中/大）
- 技术风险点

### 与 OpenClaw/阿布的结合点
- 能否做成 skill
- 能否集成到现有功能
- 需要什么新能力

---

## 📊 优先级评估
- **紧迫性**：高/中/低
- **影响力**：高/中/低
- **实现难度**：小/中/大
- **推荐优先级**：P0/P1/P2

---

## 📝 后续演进
- [ ] 待完善
- [ ] 已讨论
- [ ] 已转化为 PRD
- [ ] 开发中
- [ ] 已实现
- [ ] 搁置

**关联需求/PR**：(实现后填)

---

## 📎 附件
- 截图/链接/参考资料
```

---

## 🏷️ 标签建议

| 标签 | 说明 |
|------|------|
| #功能 | 新功能想法 |
| #技能 | 新 skill 想法 |
| #自动化 | 自动化/定时任务 |
| #AI | AI 相关能力 |
| #金融 | 股票/基金/加密货币 |
| #效率 | 提效工具 |
| #监控 | 监控/告警 |
| #集成 | 第三方服务集成 |

---

## 📂 文件命名

格式：`YYYY-MM-DD-简短标题.md`

例如：
- `2026-03-11-智能早报优化.md`
- `2026-03-15-股价预警增强.md`
