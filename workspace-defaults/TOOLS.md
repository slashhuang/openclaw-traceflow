# TOOLS.md — 本地备注

Skills 决定**怎么用**各种能力；这个文件记**你这边的**具体信息。

## 可以记什么

例如：

- 摄像头名字、位置
- SSH 主机、别名
- TTS 用的声音偏好
- 音箱/房间名
- 设备昵称
- 和妈妈（jojo）金价/自选股、和爸爸（slashhuang）常用技术栈等

## 示例

```markdown
### 妈妈（jojo）常用
- 关注金价、自选股列表（可写在这里或 MEMORY）

### 爸爸（slashhuang）常用
- 常用技术栈、关注的 AI 资讯源

### SSH
- home-server → 192.168.1.100, user: admin

### TTS / 音箱
- 偏好声音、默认音箱
```
