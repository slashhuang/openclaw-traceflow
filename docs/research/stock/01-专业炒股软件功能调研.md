## 🔑 核心功能对比

### 1. 行情监控

| 功能 | 同花顺 | 东方财富 | 富途 |
|------|--------|----------|------|
| 实时行情 | ✅ | ✅ | ✅ |
| Level-2 深度 | ✅ 付费 | ✅ 付费 | ✅ 部分免费 |
| 资金流向 | ✅ | ✅ | ✅ |
| 龙虎榜 | ✅ | ✅ | ❌ |
| 异动提醒 | ✅ | ✅ | ✅ |

**最佳实践**：
- 设置**价格异动提醒**（涨超 5%、跌超 5%）
- 关注**主力资金流向**（连续 3 日净流入）
- 监控**龙虎榜机构席位**（机构买入>卖出）

| 功能 | 同花顺 | 东方财富 | 富途 |

| 持仓盈亏分析 | ✅ | ✅ | ✅ |
