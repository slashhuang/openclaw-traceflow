## 📈 核心图表类型与最佳实践

### 1. K 线图（蜡烛图）

**最佳实践**：

| 要素 | 推荐设置 | 原因 |
|------|----------|------|
| 颜色 | 红涨绿跌（A 股） | 符合习惯 |
| 周期 | 日 K+ 周 K+60 分钟 K | 多周期共振 |
| 均线 | 5/20/60 日 | 短中长期趋势 |
| 成交量 | K 线下方柱状图 | 量价配合 |
| 复权 | 前复权 | 反映真实收益 |

**关键形态识别**：
- ✅ **金叉**：短期均线上穿长期均线（买入信号）
- ✅ **死叉**：短期均线下穿长期均线（卖出信号）
- ✅ **突破**：股价突破关键阻力位（配合成交量）
- ✅ **背离**：股价新高但指标不新高（警惕反转）

### 2. 收益曲线图

**最佳实践**：

| 要素 | 推荐设置 | 原因 |
|------|----------|------|
| 基准对比 | 沪深 300/标普 500 | 看是否跑赢大盘 |
| 时间跨度 | 1 年/3 年/5 年/成立以来 | 长期表现 |
| 累计收益 | % 显示 | 直观对比 |
| 最大回撤 | 标注在图上 | 风险评估 |
| 夏普比率 | 显示数值 | 风险调整后收益 |

**示例配置**：
```
我的组合 vs 沪深 300 vs 中证 500
时间：过去 3 年
显示：
- 累计收益曲线
- 最大回撤标注（-15%）
- 年化收益（12%）
- 夏普比率（1.2）
```

| 要素 | 推荐设置 | 原因 |
|------|----------|------|

### 4. 估值历史分位图

**最佳实践**：

| 要素 | 推荐设置 | 原因 |
|------|----------|------|
| 指标 | PE/PB/PS/股息率 | 多维度估值 |
| 历史跨度 | 10 年或成立以来 | 完整周期 |
| 分位标注 | 当前值在历史中的百分位 | 判断高估/低估 |
| 颜色警示 | <20% 分位绿色（低估），>80% 分位红色（高估） | 直观警示 |

**示例配置**：
```
沪深 300 PE 历史分位（10 年）
当前 PE：12.5x
历史分位：35%（偏低估）
建议：可以加仓

个股 PE 历史分位（5 年）
当前 PE：45x
历史分位：85%（高估）
建议：谨慎或减仓
```

### 7. 风险仪表盘

**最佳实践**：

| 指标 | 预警阈值 | 颜色 | 行动 |
|------|----------|------|------|
| 单只股票仓位 | >20% | 黄→红 | 减仓 |
| 单一行业仓位 | >40% | 黄→红 | 分散 |
| 总仓位 | >80% | 黄 | 保留现金 |
| 账户回撤 | -10% | 黄，-15% 红 | 止损 |
| VIX 波动率 | >25 黄，>30 红 | 黄→红 | 减仓 |
| 估值分位 | >80% | 红 | 谨慎 |

**示例配置**：
```
风险仪表盘
✅ 单只股票仓位：15%（正常）
✅ 单一行业仓位：35%（正常）
⚠️ 总仓位：85%（偏高，建议降至 80% 以下）
✅ 账户回撤：-5%（正常）
⚠️ VIX 波动率：28（偏高，建议减仓）
✅ 估值分位：45%（正常）

综合风险等级：中等（建议适度减仓）
```

## 🎨 设计原则（最佳实践）

### 1. 信息密度控制

| 原则 | 说明 | 示例 |
|------|------|------|
| 一屏一主题 | 每屏只展示一个主题 | K 线图单独一屏 |
| 7±2 法则 | 同时展示不超过 7 个元素 | 均线不超过 5 条 |
| 层次分明 | 重要信息突出，次要信息弱化 | 收益曲线粗，基准细 |
| 避免 clutter | 去掉不必要的装饰 | 少用 3D 效果 |

### 2. 颜色使用规范

| 场景 | 推荐颜色 | 避免 |
|------|----------|------|
| 涨跌 | 红涨绿跌（A 股） | 红绿混用 |
| 警示 | 红色（高危）、黄色（注意） | 蓝色警示 |
| 中性 | 灰色、蓝色 | 过于鲜艳 |
| 色盲友好 | 避免红绿对比 | 纯红绿区分 |

### 3. 交互设计

| 功能 | 推荐实现 | 原因 |
|------|----------|------|
| 悬停显示 | 鼠标悬停显示详细数据 | 减少 clutter |
| 下钻 | 点击行业查看个股 | 层层深入 |
| 时间选择 | 1 周/1 月/1 年/3 年/5 年 | 多周期分析 |
| 对比功能 | 添加基准对比 | 相对表现 |
| 导出 | 支持 PNG/PDF 导出 | 分享报告 |

## ⚠️ 常见错误与避免方法

| 错误 | 问题 | 最佳实践 |
|------|------|----------|
| 信息过载 | 一屏显示太多指标 | 一屏一主题，7±2 法则 |
| 颜色混乱 | 红绿蓝黄混用 | 统一配色规范 |
| 缺少对比 | 只看绝对收益 | 添加基准对比 |
| 忽视风险 | 只展示收益曲线 | 同时展示最大回撤 |
| 更新滞后 | T+1 数据 | 尽可能实时或每日更新 |
| 无法下钻 | 只有汇总数据 | 支持层层下钻 |
