## 💡 核心理念

### 股票投资的本质

```
股票价格 = 供需关系决定

供不应求 → 价格上涨 → 趋势形成 → 顺势买入
供过于求 → 价格下跌 → 趋势向下 → 回避/做空
```

**关键洞察**：
- ✅ **基本面决定长期趋势**（供需关系）
- ✅ **技术面确认中期趋势**（均线、成交量）
- ✅ **消息面影响短期波动**（催化剂）

### 3. 供需匹配分析

**核心问题**：供给和需求匹配吗？

| 供需状态 | 价格趋势 | 投资策略 | 案例 |
|----------|----------|----------|------|
| **需求↑ 供给→** | 价格↑↑ | 强烈买入 | 2023 AI 芯片（NVDA） |
| **需求↑ 供给↑** | 价格→ | 观望 | 2024 电动车（竞争激烈） |
| **需求→ 供给↑** | 价格↓ | 回避/做空 | 2024 光伏（产能过剩） |
| **需求↓ 供给→** | 价格↓↓ | 坚决回避 | 传统燃油车零部件 |
| **需求↓ 供给↓** | 价格→ | 观望 | 传统 PC（市场萎缩） |

**最佳实践**：
```
买入信号：
✅ 需求爆发 + 供给受限（最佳组合）
✅ 需求稳定 + 供给收缩（涨价逻辑）
✅ 需求拐点 + 供给滞后（周期反转）

卖出信号：
❌ 需求萎缩 + 供给扩张（最差组合）
❌ 需求拐点 + 供给过剩（周期下行）
❌ 技术被替代（长期逻辑破坏）
```

## 📈 趋势投资策略

### 1. 趋势识别

**均线系统**：

| 均线组合 | 趋势状态 | 操作 |
|----------|----------|------|
| 5 日>20 日>60 日 | 强势上涨 | 持有/加仓 |
| 5 日>20 日，20 日>60 日走平 | 震荡上行 | 持有 |
| 5 日<20 日<60 日 | 强势下跌 | 回避/卖出 |
| 均线纠缠 | 震荡 | 观望 |

**成交量确认**：
```
上涨趋势：
✅ 价格上涨 + 成交量放大（健康）
❌ 价格上涨 + 成交量萎缩（背离，警惕）

下跌趋势：
✅ 价格下跌 + 成交量放大（恐慌，可能见底）
❌ 价格下跌 + 成交量萎缩（阴跌，继续回避）
```

**趋势线**：
```
上升趋势线：连接两个以上低点
- 股价在趋势线上方 → 趋势未破，持有
- 股价跌破趋势线 → 趋势反转，卖出

下降趋势线：连接两个以上高点
- 股价在趋势线下方 → 趋势未破，回避
- 股价突破趋势线 → 趋势反转，买入
```

---

### 2. 趋势跟踪策略

**核心原则**：顺势而为，不猜顶底

```
买入条件（同时满足）：
1. 供需分析：需求↑ 供给→ 或 需求↑↑
2. 技术确认：5 日>20 日>60 日均线
3. 成交量：上涨放量
4. 催化剂：业绩/产品/政策利好

卖出条件（满足任一）：
1. 供需恶化：需求拐点/供给大幅扩张
2. 技术破位：跌破 20 日/60 日均线
3. 成交量：下跌放量
4. 止损触发：-7% 强制止损
```

**仓位管理**：
```
趋势初期：30% 仓位（试探）
趋势确认：60% 仓位（加仓）
趋势加速：80% 仓位（重仓）
趋势末端：50% 仓位（减仓）
趋势反转：0% 仓位（清仓）
```

### 3. 周期股投资策略

**核心逻辑**：供需错配→价格上涨→利润暴增→股价上涨

**典型周期**：
| 行业 | 周期长度 | 驱动因素 | 案例 |
|------|----------|----------|------|
| 内存 | 3-4 年 | 产能周期 | 美光（2023-2024 暴涨） |
| 面板 | 2-3 年 | 产能周期 | 京东方 |
| 猪肉 | 3-4 年 | 猪周期 | 牧原股份 |
| 航运 | 5-10 年 | 供需周期 | 中远海控（2021 暴涨） |
| 大宗商品 | 5-10 年 | 经济周期 | 铜、铝、石油 |

**投资时点**：
```
最佳买入：
✅ 行业亏损，产能退出（供给收缩）
✅ 需求企稳回升（需求拐点）
✅ 产品价格开始上涨（信号确认）
✅ 龙头股价率先企稳（资金认可）

最佳卖出：
❌ 行业暴利，产能扩张（供给增加）
❌ 需求增速放缓（需求拐点）
❌ 产品价格开始下跌（信号确认）
❌ 龙头股价率先下跌（资金出逃）
```

---

### 4. 成长股投资策略

**核心逻辑**：需求爆发→营收高增→利润释放→股价上涨

**典型成长**：
| 行业 | 成长驱动 | 渗透率 | 案例 |
|------|----------|--------|------|
| AI 芯片 | AI 革命 | <20% | 英伟达（2023-2024 暴涨） |
| 电动车 | 能源转型 | 30-50% | 特斯拉（2020-2021 暴涨） |
| 创新药 | 人口老龄化 | <30% | 药明康德 |
| 云计算 | 数字化转型 | 30-50% | 亚马逊 AWS |

**投资时点**：
```
最佳买入：
✅ 渗透率<30%（早期阶段）
✅ 营收增速>50%（高增长）
✅ 毛利率提升（规模效应）
✅ 行业龙头（马太效应）

最佳卖出：
❌ 渗透率>50%（成长放缓）
❌ 营收增速<20%（增长降速）
❌ 毛利率下降（竞争加剧）
❌ 估值过高（PE>100x）
```

## 🎯 实战案例

### 案例 1：美光科技（内存周期）

**供需分析**：
```
2022 年：
- 需求：手机/PC 需求下滑 ❌
- 供给：产能过剩 ❌
- 价格：内存价格暴跌 ❌
- 股价：$60 → $50（回避）

2023 年中：
- 需求：AI 服务器需求爆发 ✅
- 供给：产能收缩（厂商减产） ✅
- 价格：内存价格企稳 ✅
- 股价：$50 → $70（关注）

2023 年底：
- 需求：AI+ 手机/PC 复苏 ✅✅
- 供给：产能紧张 ✅
- 价格：内存价格上涨 ✅✅
- 股价：$70 → $120（买入）

2024 年：
- 需求：持续旺盛 ✅
- 供给：产能扩张（警惕） ⚠️
- 价格：继续上涨 ✅
- 股价：$120 → $150（持有，警惕拐点）
```

**教训**：
- ✅ 周期底部（行业亏损）→ 关注
- ✅ 供需拐点（价格企稳）→ 买入
- ✅ 趋势确认（均线多头）→ 加仓
- ⚠️ 产能扩张信号 → 警惕

## ⚠️ 风险防范

### 趋势投资陷阱

| 陷阱 | 表现 | 防范 |
|------|------|------|
| 假突破 | 突破后快速回落 | 等 3 天确认，成交量配合 |
| 趋势末端 | 加速上涨后 | 分批止盈，不追高 |
| 供需逆转 | 需求拐点/供给扩张 | 密切跟踪行业数据 |
| 黑天鹅 | 突发事件 | 严格止损，-7% 强制卖出 |

### 必须遵守的纪律

1. **供需优先**：先分析供需，再看技术面
2. **顺势而为**：不猜顶底，趋势破位再卖出
3. **严格止损**：-7% 强制止损
4. **分批操作**：建仓/加仓/止盈都分批
5. **持续跟踪**：每周跟踪供需变化

## 📝 调研总结

### 核心框架

```
1. 供需分析（基本面）
   - 供给：稀缺性、产能、竞争格局
   - 需求：增速、渗透率、替代品

2. 趋势确认（技术面）
   - 均线：5 日>20 日>60 日
   - 成交量：上涨放量
   - 趋势线：未破持有

3. 催化剂（消息面）
   - 业绩超预期
   - 产品涨价
   - 政策利好

4. 风险管理
   - 止损：-7% 强制卖出
   - 仓位：单只≤20%，总仓位≤80%
   - 跟踪：每周跟踪供需变化
```
