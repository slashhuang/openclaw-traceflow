## 🔑 核心监控策略

### 1. 趋势跟踪策略

**原理**：顺势而为，追涨杀跌（但有纪律）

```python
# 双均线策略（示例）
if 短期均线 > 长期均线 and 昨日短期均线 <= 昨日长期均线:
    买入()  # 金叉买入
elif 短期均线 < 长期均线 and 昨日短期均线 >= 昨日长期均线:
    卖出()  # 死叉卖出
```

**最佳实践**：
- **均线组合**：5 日 +20 日（短线），20 日 +60 日（中线）
- **止损**：-8% 强制止损
- **止盈**：移动止盈（最高点回撤 -10% 卖出）
- **适用**：震荡上行市场

**风险**：震荡市频繁交易，手续费高

---

### 2. 均值回归策略

**原理**：涨多了会跌，跌多了会涨

```python
# 布林带策略（示例）
if 股价 < 布林带下轨 and RSI < 30:
    买入()  # 超卖买入
elif 股价 > 布林带上轨 and RSI > 70:
    卖出()  # 超买卖出
```

**最佳实践**：
- **指标组合**：布林带 + RSI + 成交量
- **仓位**：分批建仓（30%+30%+40%）
- **止损**：-5% 止损（回归失败）
- **适用**：震荡市

**风险**：单边行情会亏损

---

### 3. 多因子选股策略

**原理**：用多个指标筛选优质股票

**常用因子**：

| 因子类型 | 具体指标 | 权重 |
|----------|----------|------|
| 价值因子 | PE、PB、股息率 | 30% |
| 成长因子 | 营收增速、净利润增速 | 30% |
| 动量因子 | 20 日涨幅、相对强度 | 20% |
| 质量因子 | ROE、毛利率、负债率 | 20% |

**最佳实践**：
- **每月调仓**：选出前 10 只股票
- **等权重配置**：每只 10% 仓位
- **止损**：单只 -10% 剔除
- **适用**：长期投资

**风险**：因子失效、风格切换

---

### 4. 网格交易策略

**原理**：震荡市自动低吸高抛

```python
# 网格策略（示例）
基准价 = 100 元
网格间距 = 5%

for 价格 in 实时价格:
    if 价格 <= 基准价 * 0.95:
        买入一格()
    elif 价格 >= 基准价 * 1.05:
        卖出一格()
```

**最佳实践**：
- **网格间距**：3%-5%（根据波动率）
- **网格数量**：10-20 格
- **底仓**：50% 仓位起步
- **适用**：震荡市（ETF、大盘股）

**风险**：单边行情破网

---

### 5. 事件驱动策略

**原理**：重大事件后快速反应

**监控事件**：
- 业绩预告（超预期→买入）
- 重大合同（中标→买入）
- 高管增持（信心→买入）
- 政策利好（行业受益→买入）

**最佳实践**：
- **快速反应**：事件发生后 5 分钟内决策
- **设置条件单**：提前预设买卖点
- **止损**：-5% 止损（事件不及预期）
- **适用**：短线交易

**风险**：信息滞后、利好出尽

## 📈 监控仪表盘设计

### 核心指标

| 指标 | 监控频率 | 预警阈值 |
|------|----------|----------|
| 账户总盈亏 | 实时 | -5% 预警，-10% 止损 |
| 单只股票盈亏 | 实时 | -7% 止损 |
| 仓位比例 | 每日 | >80% 预警 |
| 持仓集中度 | 每日 | 单只>20% 预警 |
| 市场波动率 | 每日 | VIX>30 减仓 |
| 资金流向 | 实时 | 主力连续 3 日流出预警 |

### 自动化监控

```
每 5 分钟：
  - 更新股价
  - 检查条件单触发
  - 监控异动（涨超 5%、跌超 5%）

每日收盘后：
  - 计算当日盈亏
  - 检查仓位比例
  - 更新止损止盈线
  - 扫描公告/新闻

每周日：
  - 复盘本周交易
  - 调整策略参数
  - 筛选下周标的
```

## ⚠️ 风险防范

### 常见陷阱

| 陷阱 | 表现 | 防范 |
|------|------|------|
| 过度交易 | 频繁买卖，手续费高 | 设置交易次数上限 |
| 追涨杀跌 | 情绪化交易 | 用条件单自动执行 |
| 满仓单只 | 风险集中 | 单只≤20% 仓位 |
| 不止损 | 越跌越补，深套 | 强制 -7% 止损 |
| 盲目杠杆 | 融资融券爆仓 | 杠杆≤30% 仓位 |

### 必须遵守的纪律

1. **止损纪律**：单只 -7% 强制止损，总账户 -15% 清仓
2. **仓位纪律**：单只≤20%，总仓位≤80%
3. **交易纪律**：不追涨杀跌，按计划执行
4. **复盘纪律**：每周复盘，总结得失
